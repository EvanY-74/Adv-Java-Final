# Boxed in

Boxed in is a  board game where each player tries to surround the opponent's space so that they control more territory.

Gameplay:
- Players take turns placing moves to expand their own area while restricting the other player.
- The objective is to contain the opponent and leave them with less available space.
- The winner is the player who succeeds in surrounding more of the board and leaving the other player boxed in.

Controls:
- Enter moves using board coordinates like `f7`.
- Use the same format each turn to select, move, and place a wall.

## How to run

Clone the repository

Run these commands

```
javac Main.java
java Main.java
```
